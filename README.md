Single-activity MVVM online music discovery and player app, powered by the Napster API and Firebase.
Users can discover, search and play songs, albums, and playlists by browsing through the app. 
Music data are retrieved through the Napster API while user data are held on Firebase.

Resources used: Retrofit 2.0, Exoplayer, Glide / Picasso, ConstraintLayout with support of different screen sizes, Palette API, Navigation Components

Backend: Firebase Authentication, Firestore, Cloud Storage
